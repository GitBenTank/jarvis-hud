## YouTube Pinned Comment Template

**Jarvis HUD — Secure AI Code Execution**

Repo: https://github.com/GitBenTank/jarvis-hud

What it does:
• Control plane for AI agents with file/tool access
• Propose → Approve → Execute (approval ≠ execution)
• Receipts for every action (artifact + log)
• Optional auth + step-up for execute
• Adapters: code.diff, content.publish, youtube.package, system.note

Stack: Next.js 16, React 19, TypeScript, pnpm

Run: `pnpm install && pnpm dev` → http://127.0.0.1:3000
