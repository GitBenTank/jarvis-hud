# Warm outreach — VP Eng / Head of Platform

Use when there is **existing trust** (mutual connection, portfolio overlap, prior collaboration). Personalize the bracketed lines. Keep under **180 words** in the body if possible.

---

## Email / DM (copy base)

**Subject:** [Governed agent pilot — 2 weeks, one workflow]

Hi [Name],

Good to [see you at X / catch up on Y / hear what you’re doing with Z].

We’ve been tightening **Jarvis** around a very specific wedge: **governed agent execution**—propose → **explicit approve** → execute → **receipt + trace**—on **your** infra, not a slide. I’m opening a small number of **2-week pilots** with teams that are already experimenting with agents but feel the **“who approved this action?”** gap getting real.

**Fixed scope:** one workflow you pick, one agreed environment shape, a **pilot charter** + **proof bundle** at the end so your eng (and later security) can **inspect what happened** instead of trusting model output.

If that’s remotely relevant, happy to do a **20-minute fit call** this week or next—no deck required on your side. If you’re not the right owner, who on your team is running **internal agents + production risk**?

Best,  
[Ben]

**P.S.** One-pager here if useful: [link to PDF or shared doc seeded from `governed-agent-pilot-vp-eng-one-pager.md`]
