# CTRL+STRUM / DEVHOUSE Ecosystem Tree

## Core Model

CTRL+STRUM is the publishing and media layer.

DEVHOUSE is the build, product, and builder ecosystem layer.

Ben is the operator layer connecting the two.

This separation keeps the system clear:
- DEVHOUSE builds.
- CTRL+STRUM publishes.
- Ben connects the story, execution, and relationships.

## System Flow

DEVHOUSE builds.
CTRL+STRUM publishes.
Audience discovers the work.
Some audience members become builders.
Builders join or interact with DEVHOUSE.
More projects get built.
CTRL+STRUM turns those projects into stories, videos, docs, merch, and software narratives.

Loop:
DEVHOUSE builds → CTRL+STRUM publishes → Audience grows → Builders join → DEVHOUSE grows → More builds happen
